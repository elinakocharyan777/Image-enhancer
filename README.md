# ML-модель для улучшения изображений

Проект выполнен в рамках онлайн-практики VK Education

## Описание проекта

Проект представляет собой браузерную систему для улучшения изображений в реальном времени. После загрузки изображения ML-модель анализирует его статистические характеристики (яркость, контрастность и насыщенность), подбирает оптимальные параметры коррекции и передает их алгоритму обработки изображения.

Все вычисления выполняются локально в браузере пользователя без отправки изображений на сервер. Для сохранения отзывчивости интерфейса обработка выполняется в отдельном потоке посредством Web Worker.

## Цель проекта

Разработать систему, реализующую улучшение изображений в реальном времени посредством запуска ML-модели или алгоритма подбора параметров коррекции в браузере пользователя

## Поддерживаемые форматы

На вход система принимает изображения следующих форматов:

- JPG / JPEG
- PNG
- BMP
- HEIC / HEIF

Формат HEIC / HEIF предварительно конвертируется в JPEG на стороне клиента, так как браузеры не всегда поддерживают прямое декодирование HEIC через Canvas API

Готовое изображение выгружается в формате PNG

## Логика работы системы

1. Пользователь выбирает изображение
2. Система проверяет формат файла
3. При необходимости HEIC / HEIF автоматически преобразуется в JPEG
4. Изображение отображается в интерфейсе
5. Пользователь запускает обработку
6. Создается задача обработки и формируется уникальный идентификатор taskId
7. Изображение загружается в Canvas
8. Пиксели изображения передаются в Web Worker
9. Worker извлекает статистические признаки изображения
10. ML-модель анализирует полученные признаки
11. ML-модель вычисляет оптимальные параметры коррекции.
12. Worker применяет коррекцию изображения
13. Пользователь получает автоматически улучшенное изображение
14. При необходимости пользователь дополнительно корректирует изображение с помощью ползунков
15. Готовый результат сохраняется в формате PNG.

## API модуля обработки изображений

В проекте реализован отдельный API-модуль `ImageEnhancerAPI`.

### `createTask(file)`

Создаёт задачу обработки изображения и возвращает идентификатор задачи.

### `getStatus(taskId)`

Возвращает текущий статус задачи, прогресс выполнения и информацию об ошибке, если она возникла.

### `cancelTask(taskId)`

Прерывает выполнение задачи обработки изображения.

### `getResult(taskId)`

Возвращает ссылку на готовое обработанное изображение.

### Событие `statuschange`

Возникает при изменении статуса или прогресса выполнения задачи.

### Событие `result`

Возникает после успешного завершения обработки изображения.

### Событие `cancelled`

Возникает после прерывания задачи.

### Событие `error`

Возникает при ошибке обработки.

# Работа ML-модели

В проекте используется компактная ML-модель, работающая непосредственно в браузере пользователя.

Перед обработкой из изображения извлекаются статистические признаки:

- средняя яркость;
- средняя контрастность;
- средняя насыщенность.

На основании полученных признаков модель вычисляет:

- величину изменения яркости;
- коэффициент контрастности;
- коэффициент насыщенности.

Полученные параметры автоматически передаются алгоритму обработки изображения, который применяет их к каждому пикселю изображения.

После завершения автоматического улучшения пользователь может дополнительно изменить параметры вручную при помощи элементов управления интерфейса.

## Ограничения

В проекте учтены следующие ограничения:

- обработка изображений до 15 Мп
- максимальное время обработки- до 30 секунд
- асинхронная работа без блокировки браузера
- поддержка массовых современных браузеров
- выгрузка результата в формате PNG

## Локальный запуск

При необходимости проект можно запустить локально.

1. Клонировать репозиторий:

```bash
git clone https://github.com/elinakocharyan777/Image-enhancer.git
```

2. Перейти в папку проекта:

```bash
cd Image-enhancer
```

3. Запустить локальный HTTP-сервер:

```bash
python3 -m http.server 8000
```

4. Открыть браузер:

```
http://localhost:8000
```
